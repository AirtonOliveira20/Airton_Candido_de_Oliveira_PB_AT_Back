# Sistema de Controle de Ativos de TI

## Descrição
Este projeto consiste em um sistema de controle de ativos de TI, focado principalmente nas operações de CRUD com itens, tal como cadastro, listagem, edição e exclusão. 
O objetivo principal deste CRUD é praticar a integração entre backend e frontend via API, o uso de testes automatizados e automação de build com GitHub Actions.

## Funcionalidades
- Cadastro de ativos
- Listagem de ativos
- Edição de ativos
- Exclusão de ativos
- Validação de dados
- Geração de build com Maven
- Execução de testes automatizados

## Tecnologias utilizadas
### Backend
- Java
- Spring Boot
- Maven
- JUnit
- Selenium
- H2

### Frontend
- React
- CoreUI
- JavaScript

### Automação e DevOps
- GitHub Actions

## Como executar o projeto
### Backend
1. Abrir o projeto na IDE
2. Para o banco de dados foi usado H2, através do arquivo data.sql.
3. Executar a aplicação.

### Frontend
1. Abrir a pasta do frontend no VS Code
2. Instalar as dependências:
   npm install
3. Iniciar a aplicação
   npm start
4. Abrir a aplicação no localhost:3000
